### P2: Ideation Agent ✅ 完成

**提交**: fa638fc

**实现功能**:
- **文献调研**: `search_arxiv()` + `search_semantic_scholar()`，去重处理
- **候选假设生成**: 基于文献差距生成 5 个候选想法
- **Novelty Kill Search**: 3 个搜索查询验证新颖性
- **二元可验证假设**: `hypothesis_binary` 字段包含 YES/NO 可测试声明
- **失败模式分析**: 每个想法 ≥3 个失败场景 (HIGH/MEDIUM/LOW 可能性)
- **量化成功标准**: `SuccessCriteria` 锁定实验前指标、目标值、统计阈值
- **选择逻辑**: 综合评分 = feasibility × impact × novelty，低于阈值则 REJECTED
- **输出文件**: `proposal.md` (人类可读) + `ideation_output.json` (结构化数据)

**新增模型**:
- `CandidateIdea` - 完整评估的候选想法
- `FailureMode` - 失败场景
- `SuccessCriteria` - 锁定的成功标准
- `NoveltyCheckResult` - 新颖性检查结果
- `ClosestPaper` - 最相似的已有论文

### P3-P5: Planning, Experiment, Writing Agents ✅ 完成

**提交**: 8864267

---

**P3 - Planning Agent** (`agents/planning.py`):
- 输入: `CandidateIdea` from Ideation
- 功能:
  - 生成完整实验计划 (4个步骤: setup → baseline → proposed → evaluate)
  - 指定数据集 (CIFAR-10, ImageNet-100)
  - 定义评估指标 (与 ideation 成功标准对齐)
  - 消融实验设计 (≥2 个 ablation 配置)
  - 超参数明确指定
  - 风险评估 (3+ 风险 + 缓解策略)
  - 默认 3 个随机种子 [42, 123, 456]
- 输出: `plan.json` + `plan.md`
- 可拒绝不可行计划

**P4 - Experiment Agent** (`agents/experiment.py`):
- 输入: `PlanningOutput`
- 功能:
  - 从 plan 步骤生成可执行 Python 代码
  - Docker 沙箱执行 (512MB, 120s 超时)
  - 本地执行回退
  - 步骤依赖检查 (DAG 验证)
  - 失败重试 (最多 3 次)
  - 多随机种子运行
  - 结果聚合: mean ± std
  - 统计显著性检验 (t-test, p-value)
- 输出: `report.json` + `report.md`

**P5 - Writing Agent** (`agents/writing.py`):
- 输入: 所有上游产物 (ideation + planning + experiment)
- 功能:
  - 生成完整 LaTeX 论文 (ACL/ICML 风格)
  - 章节: Abstract, Introduction, Related Work, Methodology, Experiments, Results, Discussion, Conclusion
  - 数字验证循环: 每个数字必须来自 report.json
  - 引用管理 (BibTeX)
  - 结果表格 (含 mean±std, 显著性标记)
  - LaTeX 编译 (如 pdflatex 可用)
- 输出: `paper.tex` + `paper.pdf` + `paper.md`

**新增模型**:
- `AblationStudy`, `PlanningOutput` 扩展 (plan.py)
- `StepResult`, `ExperimentResult`, `ExperimentOutput` (experiment.py)
- `Citation`, `PaperDraft`, `WritingOutput` (paper.py)

**测试覆盖**:
- `tests/test_planning.py` - 计划生成、消融设计、拒绝逻辑
- `tests/test_experiment.py` - 代码生成、步骤执行、重试逻辑、结果聚合
- `tests/test_writing.py` - 论文结构、数字验证、引用一致性
